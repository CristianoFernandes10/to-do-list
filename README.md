Lista de Tarefas

Projeto simples que fiz para treinar JavaScript na prática.
A ideia é bem direta — adicionar, editar, concluir e remover tarefas.




Funcionalidades

Adicionar tarefa
O texto é formatado e validado antes de entrar na lista.

Evita duplicadas
Não deixa criar duas tarefas com o mesmo texto.

Editar tarefa
Troca o texto da tarefa direto na lista.
(E se a tarefa já estiver concluída, o botão de editar fica desativado.)

Marcar como concluída / desfazer
Usa toggle para alternar entre concluída e não concluída.

Remover tarefa
Some da lista e do localStorage.

Salva tudo no navegador
Quando recarrega a página, tudo volta direitinho.





Tecnologias usadas

HTML

CSS

JavaScript puro

LocalStorage
